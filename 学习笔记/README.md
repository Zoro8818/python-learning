# 学习笔记

本目录用于保存整理后的 Python 学习笔记，适合后续上传 GitHub 展示。

## 目录说明

```text
学习笔记/
  README.md
  课堂小结/
    第16节_字符串拼接与格式化.md
    第17节_input输入与类型转换_阶段小结.md
    ...
  images/
    数据容器类型.png
    列表的定义及访问.png
    课堂小结/
      第16节_字符串格式化小结.png
      第17节_input输入与类型转换_阶段小结.png
      ...
```

## 使用规则

- `.md` 文件用于保存整理后的文字笔记。
- `课堂小结/` 用于保存课堂小结的文字版 Markdown。
- `images/` 用于保存笔记中引用的图片和截图。
- `images/课堂小结/` 用于保存课堂小结配套图片。
- 临时草稿、每日流水账、课堂过程记录不要放在这里，优先放到内部学习记录区。
- 图片不要直接堆在 `学习笔记/` 根目录。
