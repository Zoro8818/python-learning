## 1. 比较运算符的作用

比较运算符也称为关系运算符，用于比较两个值之间的关系。

比较运算符会计算左右两边的表达式，并返回一个布尔值：

```text
True  表示比较关系成立
False 表示比较关系不成立
```

## 2. 常见比较运算符

| 运算符 | 描述 | 示例 | 含义 |
| --- | --- | --- | --- |
| `==` | 等于 | `a == b` | 判断 `a` 是否等于 `b` |
| `!=` | 不等于 | `a != b` | 判断 `a` 是否不等于 `b` |
| `>` | 大于 | `a > b` | 判断 `a` 是否大于 `b` |
| `>=` | 大于等于 | `a >= b` | 判断 `a` 是否大于等于 `b` |
| `<` | 小于 | `a < b` | 判断 `a` 是否小于 `b` |
| `<=` | 小于等于 | `a <= b` | 判断 `a` 是否小于等于 `b` |

## 3. 重要注意事项

`=` 和 `==` 不一样。

```python
num = 10
```

表示把 `10` 赋值给变量 `num`。

```python
num == 10
```

表示判断 `num` 是否等于 `10`，结果是 `True` 或 `False`。

## 4. 示例

```python
a = 10
b = 20

print(a == b)
print(a != b)
print(a > b)
print(a >= b)
print(a < b)
print(a <= b)
```

## 5. 当前重点

- 比较运算符的结果是布尔值：`True` 或 `False`。
- `==` 是判断是否相等。
- `=` 是赋值，不是比较。
- 比较运算符是后续学习 `if` 条件判断的重要基础。
