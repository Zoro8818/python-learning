# 第30节 for循环语句

## 本节主题
学习 `for` 循环的基本语法、`else` 可选结构，以及 `for` 循环和 `while` 循环的使用场景区别。

## 1. for 循环的语法格式

```python
for 元素 in 待处理数据集:
    循环体代码（对元素进行处理）
else:
    不满足循环条件时，执行的操作
```

说明：
- `for` 用来从一个待处理数据集中，依次取出每一个元素。
- 每取出一个元素，就执行一次缩进后的循环体代码。
- `else` 是可选部分，用来表示循环正常结束后执行的操作。

## 2. for 循环的执行逻辑

1. 从待处理数据集中取出第 1 个元素。
2. 把这个元素临时保存到 `for` 后面的变量中。
3. 执行循环体代码，对当前元素进行处理。
4. 继续取下一个元素，重复执行循环体。
5. 当数据集中的元素都被处理完后，循环结束。
6. 如果写了 `else`，并且循环正常结束，就会执行 `else` 中的代码。

## 3. for 循环和 while 循环的使用场景

### while 循环

`while` 循环用于在某个条件满足时一直循环。

特点：
- 循环次数通常是未知的。
- 更关注循环开始和结束的条件。
- 适合“只要条件成立，就继续执行”的场景。

示例：
```python
while 条件表达式:
    条件成立时重复执行的代码
```

### for 循环

`for` 循环用于对一个已知的数据集进行遍历，或者进行已知次数的循环。

特点：
- 循环对象通常是已知的数据集。
- 更关注遍历每一个元素。
- 适合“把一组数据逐个处理一遍”的场景。

示例：
```python
for item in data:
    print(item)
```

## 4. 示例

```python
names = ['张三', '李四', '王五']

for name in names:
    print(name)
else:
    print('所有名字都遍历完了')
```

执行过程：
- 第一次取出 `'张三'`，执行打印。
- 第二次取出 `'李四'`，执行打印。
- 第三次取出 `'王五'`，执行打印。
- 所有元素处理完后，循环正常结束，执行 `else`。

## 5. 本节核心

`for` 循环适合遍历已知的数据集，重点是“依次处理每一个元素”；`while` 循环适合根据条件反复执行，重点是“条件是否继续成立”。
