## 1. 算术运算符的作用

算术运算符用于执行基本的数学运算，作用在一个或多个操作数上，并产生一个计算结果。

## 2. 常见算术运算符

| 运算符 | 描述 | 示例 | 结果说明 |
| --- | --- | --- | --- |
| `+` | 加法 | `1 + 5` | 1 加上 5 |
| `-` | 减法 | `8 - 3` | 8 减 3 |
| `*` | 乘法 | `3 * 8` | 3 乘以 8 |
| `/` | 除法 | `10 / 5` | 10 除以 5，结果通常是小数 |
| `//` | 整除 | `10 // 3` | 10 整除 3，结果为整数 |
| `%` | 取余/求模 | `10 % 3` | 10 除以 3 后取余数，结果为 1 |
| `**` | 幂指数 | `10 ** 3` | 10 的 3 次方 |

## 3. 当前重点

- `/` 是普通除法，结果通常是小数。
- `//` 是整除，只保留整数部分。
- `%` 是取余数，可以理解为“除完以后剩下多少”。
- `**` 是幂运算，例如 `2 ** 3` 表示 2 的 3 次方。
- 涉及浮点数的算术运算时，可能会损失精度。原因是计算机底层基于二进制存储和处理数据，二进制无法精确表示所有小数。

## 4. 练习建议

```python
a = 10
b = 3

print(a + b)
print(a - b)
print(a * b)
print(a / b)
print(a // b)
print(a % b)
print(a ** b)
```

## 5. 可做的小练习

写一个简单的“四则运算练习器”：

```python
num1 = int(input("请输入第一个整数："))
num2 = int(input("请输入第二个整数："))

print(f"{num1} + {num2} = {num1 + num2}")
print(f"{num1} - {num2} = {num1 - num2}")
print(f"{num1} * {num2} = {num1 * num2}")
print(f"{num1} / {num2} = {num1 / num2}")
```
