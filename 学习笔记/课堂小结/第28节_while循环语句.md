# 第28节 while循环语句

## 本节主题
学习 `while` 循环的基本语法、执行逻辑和注意事项。

## 1. while 循环的语法格式

```python
while 条件表达式:
    条件成立时，执行的操作（循环体语句）
    条件成立时，执行的操作（循环体语句）
    ......
else:
    循环正常结束时，执行的操作
```

说明：
- `while 条件表达式:` 表示只要条件成立，就反复执行下面缩进的循环体。
- 循环体必须缩进，通常使用 4 个空格。
- `else` 是可选部分，用来表示循环正常结束后执行的代码。

## 2. while 循环的执行逻辑

1. 先判断 `while` 后面的条件表达式。
2. 如果条件结果是 `True`，执行循环体。
3. 循环体执行完后，再回到 `while` 条件处重新判断。
4. 如果条件结果变成 `False`，循环结束。
5. 如果写了 `else`，并且循环是正常结束，就会执行 `else` 里的代码。

## 3. 注意事项

- 条件表达式的结果必须能判断为布尔类型，即 `True` 或 `False`。
- 通过空格缩进表示层级关系，循环体必须和 `while` 保持正确缩进。
- 必须提前规划好循环结束的条件，否则程序可能进入无限循环，也叫死循环。
- 循环体里通常要修改某个变量，让条件最终有机会变成 `False`。

## 4. 示例

```python
i = 1

while i <= 5:
    print('正在学习 Python')
    i += 1
else:
    print('循环正常结束')
```

执行过程：
- `i` 从 1 开始。
- 只要 `i <= 5` 成立，就执行打印语句。
- 每执行一次，`i += 1` 让 `i` 增加 1。
- 当 `i` 变成 6 时，条件不成立，循环结束。
- 最后执行 `else` 中的 `循环正常结束`。

## 5. 本节核心

`while` 循环适合处理“只要条件成立，就重复执行”的场景。写 `while` 时最重要的是三件事：条件表达式、循环体缩进、循环终止条件。
