# 第 24 节小结：if...else 语句

这一节主要学习了 Python 里的 `if...else` 条件判断结构。

## 1. if...else 语句的语法

```python
if 要判断的条件:
    条件成立时，执行对应的操作1
else:
    条件不成立时，执行对应的操作2
```

`if...else` 用来处理两种情况：

- 条件成立：执行 `if` 下面缩进的代码块。
- 条件不成立：执行 `else` 下面缩进的代码块。

## 2. if...else 语句注意事项

1. `if` 后面需要写判断条件，判断条件后面要加英文冒号 `:`。
2. `else` 后面不需要再写条件，直接写 `else:`。
3. 当 `if` 条件不成立时，程序会自动执行 `else` 代码块。
4. `if` 代码块和 `else` 代码块都需要缩进，建议缩进 4 个空格。

## 3. 示例代码

```python
age = 17

if age >= 18:
    print("你已经成年了")
else:
    print("你还未成年")
```

## 4. 本节核心

`if...else` 适合处理“二选一”的判断场景：条件成立走一条路，条件不成立走另一条路。
