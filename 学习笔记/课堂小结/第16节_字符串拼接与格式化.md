## 第 16 节：字符串拼接与字符串格式化

归档日期：2026-06-11

### 1. 字符串拼接

字符串拼接使用 `+` 号。

```python
name = "张三"
message = "你好，" + name
print(message)
```

注意：字符串不能直接和非字符串类型拼接。

```python
age = 18
print("年龄是：" + age)  # 错误
```

如果要拼接数字、布尔值等其他类型，需要先转换成字符串：

```python
age = 18
print("年龄是：" + str(age))
```

### 2. `%` 字符串格式化

基本语法：

```python
"%s" % 变量
```

多个变量：

```python
"%s..%s" % (变量1, 变量2)
```

示例：

```python
name = "张三"
age = 18
print("姓名：%s，年龄：%s" % (name, age))
```

### 3. `f` 字符串格式化

基本语法：

```python
f"内容{变量/表达式}"
```

示例：

```python
name = "张三"
age = 18
print(f"姓名：{name}，年龄：{age}")
```

也可以直接写表达式：

```python
price = 10
count = 3
print(f"总价：{price * count}")
```

### 4. 本节重点

- `+` 可以拼接字符串。
- 字符串不能直接和数字等非字符串类型拼接。
- 非字符串类型需要使用 `str()` 转换后再拼接。
- `%` 可以做字符串格式化，但写法相对旧一些。
- `f"..."` 格式化字符串更直观，推荐优先使用。
