# 第32节 嵌套循环

## 本节主题
学习嵌套循环的语法结构。嵌套循环就是在一个循环体内部，再写一个新的循环。

## 1. 什么是嵌套循环

嵌套循环指的是：外层循环每执行一次，内层循环都会完整执行一轮。

常见结构：
- 外层循环：控制大的循环次数或外层数据。
- 内层循环：在外层循环的每一次执行过程中，再对另一组数据进行循环处理。

## 2. for 嵌套循环语法

```python
# for 循环结构
for 元素 in 待处理数据集1:
    循环体的代码1
    循环体的代码2
    ...

    for 元素 in 待处理数据集2:
        循环体的代码1
        循环体的代码2
        ...

    ...
```

说明：
- 第一个 `for` 是外层循环。
- 第二个 `for` 写在外层循环的循环体内部，是内层循环。
- 内层循环需要比外层循环多缩进一层。
- Python 通过缩进判断代码属于外层循环还是内层循环。

## 3. 执行逻辑

1. 外层循环先取出一个元素。
2. 执行外层循环体中的代码。
3. 遇到内层循环时，内层循环会从头到尾完整执行一遍。
4. 内层循环结束后，继续执行外层循环剩余代码。
5. 外层循环进入下一轮时，内层循环又会重新完整执行一遍。

## 4. 示例

```python
for i in range(3):
    print('外层循环：', i)

    for j in range(2):
        print('内层循环：', j)
```

执行理解：
- 外层循环 `range(3)` 会执行 3 次。
- 每次外层循环执行时，内层循环 `range(2)` 都会执行 2 次。
- 所以内层循环总共执行 `3 * 2 = 6` 次。

## 5. 注意事项

- 嵌套循环的核心是缩进，缩进错了，层级关系就会错。
- 外层循环每执行一次，内层循环都会完整跑一遍。
- 嵌套层级不要过多，否则代码会变难读。
- 写嵌套循环时，要先想清楚：外层负责什么，内层负责什么。

## 6. 本节核心

嵌套循环就是“循环里面再写循环”。理解它时抓住一句话：外层循环每执行一次，内层循环就完整执行一轮。
