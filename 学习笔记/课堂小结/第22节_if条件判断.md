# 第 22 节小结：if 条件判断

这一节主要学习了 Python 里的 `if` 条件判断。

## 1. if 条件判断的基本格式

```python
if 判断条件:
    条件成立时执行的代码
```

如果还要处理“条件不成立”的情况，可以写：

```python
if 判断条件:
    条件成立时执行的代码
else:
    条件不成立时执行的代码
```

## 2. if 语句的注意事项

1. `if` 后面的判断条件结果必须是布尔类型，也就是 `True` 或 `False`。
2. 判断条件后面一定要写英文冒号 `:`。
3. `if` 里面要执行的代码必须缩进，通常缩进 4 个空格，用缩进表示这段代码属于 `if`。

## 3. 示例代码

```python
age = 18

if age >= 18:
    print("你已经成年了")
else:
    print("你还未成年")
```

## 4. 本节核心

程序可以根据条件是否成立，选择执行不同的代码。
