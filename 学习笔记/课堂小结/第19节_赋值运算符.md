## 1. 赋值运算符的作用

赋值运算符用于将值或表达式的结果保存到变量中。

可以理解为：

```python
变量 = 值
```

例如：

```python
num = 1 + 2
```

表示把 `1 + 2` 的结果赋值给变量 `num`，所以 `num` 的值是 `3`。

## 2. 常见赋值运算符

| 运算符 | 描述 | 等价写法 |
| --- | --- | --- |
| `=` | 赋值运算符 | `num = 1 + 2` |
| `+=` | 加法赋值运算符 | `num += 2` 等价于 `num = num + 2` |
| `-=` | 减法赋值运算符 | `num -= 2` 等价于 `num = num - 2` |
| `*=` | 乘法赋值运算符 | `num *= 2` 等价于 `num = num * 2` |
| `/=` | 除法赋值运算符 | `num /= 2` 等价于 `num = num / 2` |
| `%=` | 取模赋值运算符 | `num %= 2` 等价于 `num = num % 2` |
| `//=` | 取整除赋值运算符 | `num //= 2` 等价于 `num = num // 2` |
| `**=` | 幂赋值运算符 | `num **= 2` 等价于 `num = num ** 2` |

## 3. 重要注意事项

复合赋值运算符本身不能被拆开。

正确写法：

```python
num += 2
num //= 2
num **= 2
```

错误写法：

```python
num + = 2
num / /= 2
num * *= 2
```

说明：

- `+=`、`-=`、`*=`、`/=`、`%=`、`//=`、`**=` 是一个整体。
- 运算符中间不能加空格，也不能插入任何其他符号。
- 运算符两侧可以加空格，例如 `num += 2` 是推荐写法。
