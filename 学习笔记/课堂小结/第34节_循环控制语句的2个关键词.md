# 第34节 循环控制语句的2个关键词

## 本节主题
学习循环控制语句中的两个关键词：`break` 和 `continue`。

这两个关键词都不能单独书写，只能出现在循环中，用来控制循环的执行流程。

## 1. break 关键字的作用

`break` 的作用是：结束当前循环，并跳出循环。

特点：
- 不能单独书写。
- 只能出现在循环中。
- 表示“结束、跳出”的意思。
- 一旦执行到 `break`，当前循环会立刻停止，后面的循环次数不再继续执行。

示例：
```python
for i in range(1, 6):
    if i == 3:
        break
    print(i)
```

输出：
```text
1
2
```

解释：
- 当 `i` 等于 3 时，执行 `break`。
- 循环直接结束，所以 3、4、5 都不会再打印。

## 2. continue 关键字的作用

`continue` 的作用是：中断本次循环，直接进入下一次循环。

特点：
- 不能单独书写。
- 只能出现在循环中。
- 表示“跳过本次，继续下一次”的意思。
- 一旦执行到 `continue`，本轮循环中它后面的代码不会执行，但循环本身不会结束。

示例：
```python
for i in range(1, 6):
    if i == 3:
        continue
    print(i)
```

输出：
```text
1
2
4
5
```

解释：
- 当 `i` 等于 3 时，执行 `continue`。
- 本次循环被跳过，所以 3 不打印。
- 循环继续进入下一次，所以 4、5 仍然会打印。

## 3. break 和 continue 的区别

| 关键词 | 作用 | 结果 |
|---|---|---|
| `break` | 结束整个循环 | 后面的循环次数都不执行 |
| `continue` | 结束本次循环 | 直接进入下一次循环 |

## 4. 本节核心

`break` 是“直接结束整个循环”；`continue` 是“跳过本次，继续下一次”。两者都只能写在循环里面，不能单独使用。
