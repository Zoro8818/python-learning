# 第 25 节内容：if...elif...else 语句

这一节学习了 `if...elif...else` 多分支判断结构。

## 1. 使用场景

当一个问题有多个判断结果时，可以使用 `if...elif...else`。

例如：根据用户输入的数字，判断这个数字是正数、负数，还是 0。

- 如果数字 `> 0`：正数
- 如果数字 `< 0`：负数
- 否则：0

## 2. if...elif...else 语法结构

```python
if 要判断的条件1:
    条件1成立时，执行对应的操作1
elif 要判断的条件2:
    条件2成立时，执行对应的操作2
else:
    条件1和条件2都不成立时，执行对应的操作3
```

## 3. 示例代码

```python
num = int(input("请输入数字: "))

if num > 0:
    print(f"{num} 是 正数")
elif num < 0:
    print(f"{num} 是 负数")
else:
    print(f"{num} 是 0")
```

## 4. 执行顺序

程序会从上往下依次判断：

1. 先判断 `if` 条件。
2. 如果第 1 个条件成立，就执行 `if` 代码块，后面的 `elif` 和 `else` 不再判断。
3. 如果第 1 个条件不成立，才会继续判断 `elif` 条件。
4. 如果 `if` 和 `elif` 都不成立，最后才会执行 `else`。

## 5. 注意事项

1. `if` 和 `elif` 后面都需要写判断条件。
2. `else` 后面不需要写判断条件。
3. `if`、`elif`、`else` 后面都要写英文冒号 `:`。
4. 每个分支下面的代码块都需要缩进，建议缩进 4 个空格。

## 6. 本节核心

`if...elif...else` 适合处理“多个结果选一个”的判断场景。只要前面的某个条件成立，后面的分支就不会再执行。

## 7. 课堂小结

### 作用

`if...elif...else` 可以完成多个条件的判断，也叫多分支判断。

### 注意事项

1. `elif` 可以写多个，用来判断多个不同条件。
2. `else` 可写可不写，如果写了，必须放在最后。
3. 多个条件判断是有顺序的，会从上到下依次判断。
4. 如果前面的某个条件成立，后面的条件就不会再判断。

### 一句话记忆

`if...elif...else` 就是“多个条件按顺序判断，谁先成立就执行谁”。
