## 1. 逻辑运算符的作用

逻辑运算符用于连接多个条件表达式。

条件表达式的结果通常是布尔值：

```text
True  表示条件成立
False 表示条件不成立
```

逻辑运算符会根据多个条件的结果，返回最终的布尔结果。

## 2. 生活场景理解

### 多个条件同时成立：`and`

例如买电脑时要求：

```text
10000以内 and 大品牌 and 高配置
```

这表示多个条件必须同时满足，才符合要求。

### 多个条件满足一个即可：`or`

例如点外卖时要求：

```text
评分最高 or 配送最快
```

这表示多个条件中只要有一个满足，就符合要求。

## 3. 常见逻辑运算符

| 运算符 | 描述 | 规则 |
| --- | --- | --- |
| `and` | 逻辑与 / 并且 | 左右两边都为 `True`，结果才为 `True` |
| `or` | 逻辑或 / 或者 | 左右两边只要有一个为 `True`，结果就是 `True` |
| `not` | 逻辑非 / 取反 | `True` 变 `False`，`False` 变 `True` |

## 4. 示例

```python
a = 10
b = 20
c = 30

print(a < b and b < c)  # True
print(a > b or b < c)   # True
print(not a < b)        # False
```

## 5. 当前重点

- `and`：多个条件必须全部成立。
- `or`：多个条件只要有一个成立即可。
- `not`：对布尔结果取反。
- 逻辑运算符通常和比较运算符一起使用。
- 这是后续学习 `if` 多条件判断的基础。

## 6. 本节小结

逻辑运算符用于连接多个布尔表达式，也就是结果为 `True` 或 `False` 的表达式，并返回最终的一个布尔值。

本节需要记住三个逻辑运算符：

- `and`：并且，同时成立。
- `or`：或者，满足其一。
- `not`：非，取反操作。

## 7. 练习建议

判断一个年龄是否在 18 到 60 岁之间：

```python
age = int(input("请输入年龄："))

print(age >= 18 and age <= 60)
```

判断一个分数是否低于 60 或高于 100：

```python
score = int(input("请输入分数："))

print(score < 60 or score > 100)
```
