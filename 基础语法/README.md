# 基础语法

本目录用于保存 Python 基础语法阶段的练习代码，适合按编号顺序学习和复习。

## 学习目标

- 掌握 Python 的基本写法和运行流程。
- 熟悉输入、输出、变量、数据类型、运算符、条件判断和循环。
- 为后续学习列表、字典、函数、文件操作、办公自动化和 AI 工具开发打基础。

## 使用方法

建议每个文件按下面顺序复习：

```text
1. 先看文件名，明确主题
2. 再读注释，理解这个知识点解决什么问题
3. 运行代码，观察输出
4. 修改 1 到 2 个变量或条件，确认自己理解了变化
5. 用自己的话总结输入、处理、输出
```

## 命名规则

本目录 Python 文件统一使用：

```text
编号_主题.py
```

示例：

```text
01_字面量与变量.py
12_数据容器_list.py
```

规则：

- 编号使用两位数字，方便排序。
- 文件名不要带 `├──`、`│` 等目录树符号。
- 中文和英文混合时，用 `_` 分隔。
